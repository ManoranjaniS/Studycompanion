# 📚 StudyCompanion - Your Personal Study Assistant

**StudyCompanion** is a productivity-focused web app built to help students organize their study life. From tracking tasks and scheduling sessions to staying focused with a Pomodoro timer, this app is your ultimate study buddy.


## 🌟 Features

- ✅ Task Manager for daily/weekly study goals
- ⏳ Pomodoro Timer for focused study sessions
- 📅 Customizable Study Planner & Timetable
- 📈 Progress Tracker with visual analytics
- 📝 Quick Notes for capturing ideas/doubts
- 🌓 Dark/Light Mode toggle for better UX

## 🛠 Tech Stack

**Frontend**: `React.js` / `Thymeleaf`  
**Backend**: `Spring Boot`, `REST API`  
**Database**: `MySQL` 
**Others**: `JPA`, `Lombok`
